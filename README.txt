SpellBound - ITSE 1450 - 4/15/2019

Team Members: Danielle Whyte

SpellBound is a computer game aimed at beginning English as a second language students. A definition will appear on the screen and the player inputs their attempt at spelling the word. The program returns weather or not the word was spelled correctly.

SpellBound.java - Source code
